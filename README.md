# kusagame_pattern

愛亀であるクサガメの「かめちゃん」の、顔の横の柄をイメージした模様を生成するプログラムです。

## アルゴリズム（概要）

このプログラムは、Gray-Scott 反応拡散モデルで模様を生成します。  
初期状態にランダムなパッチとノイズを与え、時間発展によって縞・斑点パターンを作ります。  
計算は FFT を用いた IMEX スキームで進めており、比較的高速にシミュレーションできます。  
`seed` を変えると初期配置が変わり、異なる柄が得られます。
